## Backend ядра учёта СНТ (FastAPI + SQLAlchemy)

Этот backend реализует 1С‑подобную пересчитываемую модель учёта для СНТ.

- Ядро: PostgreSQL + бизнес‑логика на Python (FastAPI + SQLAlchemy).
- Все инварианты и целостность данных обеспечиваются структурой БД (FK, CHECK, транзакции).
- UI не является источником истины; backend и БД должны работать автономно.

Структура проекта будет дополняться по мере реализации схемы БД, сервисов проведения и REST‑API.
