## 🚀 Запустить проект

### Один командой (Windows)

```powershell
npm run dev
```

Открой: **http://localhost:5173**

**Логин:** `admin` / `admin` (или данные из `backend/.env`)

### Вручную (если нужно)

```powershell
# Терминал 1 — Backend
cd backend
.\venv\Scripts\Activate.ps1
uvicorn app.main:app --reload

# Терминал 2 — Frontend
cd frontend
npm run dev
```
