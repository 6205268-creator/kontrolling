# Obsidian — дубликаты для просмотра

Эта папка — копия ключевых материалов в формате Markdown для удобного просмотра в [Obsidian](https://obsidian.md). Mermaid-диаграммы отображаются в Obsidian без доп. настроек.

## Как использовать

**Вариант 1.** Открыть папку проекта как хранилище (vault) в Obsidian:  
*Файл → Открыть папку как хранилище* → выберите `e:\PROGECTS\kontrolling`. Тогда заметки из `obsidian/` и остальные `.md` будут в графе знаний.

**Вариант 2.** Скопировать папку `obsidian/` в своё существующее хранилище Obsidian — все диаграммы будут в одном месте.

## Файлы

- **КОНТРОЛЛИНГ — диаграммы.md** — все C4-диаграммы (акторы, системы, БД, контекст L1, контейнеры L2).

Исходники диаграмм в репозитории: `docs/architecture/common/*.mmd`, `docs/architecture/diagrams.md`, `domains/bank_statements/container-diagram.mmd`.
