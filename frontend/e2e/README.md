# E2E Тесты (Playwright)

## Установка

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Установите браузеры:
   ```bash
   npx playwright install
   ```

## Запуск тестов

### Запуск всех тестов
```bash
npm run test:e2e
```

### Запуск в режиме UI (интерактивный)
```bash
npm run test:e2e:ui
```

### Запуск в браузере (headed mode)
```bash
npm run test:e2e:headed
```

### Запуск конкретного теста
```bash
npm run test:e2e -- test_full_flow.spec.ts
```

### Запуск с отчётом
```bash
npm run test:e2e -- --reporter=html
npx playwright show-report
```

## Структура тестов

### Сценарий 1: Полный флоу
- Логин в систему
- Создание участка
- Создание начисления
- Регистрация платежа
- Проверка баланса

### Сценарий 2: Навигация и авторизация
- Редирект на login при отсутствии аутентификации
- Выход из системы
- Навигация по всем разделам

### Сценарий 3: Создание сущностей
- Создание владельца
- Просмотр счётчиков
- Просмотр расходов

## Требования

- Backend должен быть запущен на `http://localhost:8000`
- Frontend должен быть запущен на `http://localhost:5173`
- Тестовые данные создаются автоматически через фикстуры

## Тестовые данные

По умолчанию тесты используют следующие данные:
- Username: `testuser`
- Password: `testpassword123`

## Отчётность

После запуска тестов с `--reporter=html` откройте отчёт:
```bash
npx playwright show-report
```

## Отладка

Для отладки тестов используйте:
```bash
npm run test:e2e -- --debug
```

Или добавьте `await page.pause()` в коде теста для остановки выполнения.
