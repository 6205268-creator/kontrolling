  - SQLAlchemy 2.0 с async поддержкой + asyncpg
  - Vue 3 + TypeScript + Vite + Pinia (официальные рекомендации)
  - Pydantic v2 для валидации схем

**Решение по стеку:**
- **Первоначальный дизайн:** Django 5.x + React 18
- **Обновлённый стек:** FastAPI + Vue 3 (более современный и эффективный)
- Причина смены: FastAPI — async-first, быстрее, легче для API; Vue 3 — современнее, проще для SPA

### 2. Исследование лучших практик

**Изучено:**
- **FastAPI Best Practices 2026:**
  - Модульная структура проектов (APIRouter для разделения логики)
  - Layered Architecture (Router → Service → Repository)
  - Async/await паттерны для неблокирующего I/O
  - Dependency Injection для тестируемости

- **SQLAlchemy 2.0 Async:**
  - `create_async_engine` + `AsyncSession`
  - asyncpg драйвер для PostgreSQL
  - Интеграция с Pydantic v2 через `ConfigDict(from_attributes=True)`

- **Vue 3 + TypeScript:**
  - `<script setup>` синтаксис с TypeScript
  - Composition API для реактивности
  - Pinia вместо Vuex (официальная рекомендация)
  - Vite для сборки (замена Vue CLI)

- **Тестирование:**
  - pytest + pytest-asyncio для backend
  - TestClient → AsyncClient для async тестов
  - Vitest для frontend
  - Test pyramid approach (unit → API → E2E)

- **JWT Security:**
  - PyJWT + passlib[bcrypt] для аутентификации
  - OAuth2PasswordBearer для токенов
  - Правильное хранение секретов в environment variables
