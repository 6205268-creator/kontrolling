## Как работать с хроникой

- Открой `life-year-matrix.md` — это единственный источник правды по событиям.
- Статус каждого события: ✅ реализовано / ⚠️ частично / ❌ не реализовано / 🔍 на пересмотр.
- В конце файла — **сводная таблица бэклога** по приоритетам.
- При изменениях в коде — обновляй статусы в соответствующих строках.
- 🔍 «На пересмотр» — не выдумывать, уточнять у председателя или казначея.
