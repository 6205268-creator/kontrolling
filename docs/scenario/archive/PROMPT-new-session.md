## Готовые фразы — вставь под промпт одной строкой

- **Только матрица (явно):** `Задача: аудит EVT-001…024, обнови статусы в life-year-matrix.md по коду.`
- **Новые события:** `Задача: добавь в матрицу строки EVT-025+ по моему тексту: …`
- **Скрипт / раннер (то, о чём мы договаривались):** `Задача: Фаза B — начни реализацию раннера по §7 TECHNICAL-SPEC (скелет, профиль validate + smoke, без полного года).`
- **Не сценарий:** `Задача не про сценарий — работаем по docs/plan/current-focus.md, раздел …`
