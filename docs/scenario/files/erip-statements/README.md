# Выписки ЕРИП — имитация банковских документов

Каждый файл = одна выписка, которую казначей получает от банка (раз в неделю или чаще).  
Казначей вносит данные в систему вручную — или загружает файл, если появится автоимпорт.

## Формат файла

```
Дата операции          — когда плательщик оплатил
Дата зачисления        — когда деньги пришли на счёт СТ (обычно следующий рабочий день)
Номер операции ЕРИП    — уникальный ID из банка
Наименование плательщика — кто платил (может быть не владелец участка!)
Банк плательщика       — разные банки, все через ЕРИП
Назначение платежа     — что написал плательщик (свободный текст, бывает неинформативно)
Сумма операции (BYN)   — сколько плательщик заплатил
Комиссия ЕРИП (BYN)    — комиссия банка/ЕРИП (удерживается из суммы)
Сумма зачисления (BYN) — сколько фактически пришло на счёт СТ
```

**Комиссия ЕРИП:** 0,8% от суммы (условно — у каждого банка может быть своя ставка).

## Что интересно для разработки системы

- Плательщик ≠ владелец участка (файлы 04-10, 06-28, 08-15, 10-25)
- Назначение платежа — свободный текст, нет стандарта
- Один платёж покрывает несколько участков (05-20, 11-12)
- Неопознанный плательщик (04-22, 10-05)
- Комиссия ЕРИП: СТ получает меньше, чем заплатил член
- Дата операции ≠ дата зачисления (выходные, праздники)
