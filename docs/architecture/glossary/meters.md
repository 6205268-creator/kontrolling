# Глоссарий домена «Приборы учёта (Meters)»

Термины, связанные с учётом счётчиков воды и электроэнергии и их показаний в СТ.

| Термин | Определение | Пример использования в СТ |
|--------|-------------|----------------------------|
| **Прибор учёта (Meter)** | Счётчик воды или электроэнергии, привязанный к владельцу (Owner). При создании прибора автоматически создаётся FinancialSubject типа WATER_METER или ELECTRICITY_METER для начислений и платежей. | Счётчик воды № 12345 на участке Иванова; по нему начисляют плату за воду. |
| **Тип прибора (meter_type)** | Вид счётчика: WATER (вода) или ELECTRICITY (электроэнергия). Определяет тип связанного FinancialSubject и вид тарификации. | WATER — расчёт по воде, ELECTRICITY — по электроэнергии. |
| **Серийный номер (serial_number)** | Заводской или учётный номер счётчика; используется для идентификации и в коде FinancialSubject (FS-M-xxx). | Счётчик с serial_number 12345-67. |
| **Дата установки (installation_date)** | Дата установки или постановки прибора на учёт. Опционально. | installation_date 01.06.2020 — с этой даты ведутся показания. |
| **Показание (MeterReading)** | Зафиксированное значение счётчика на дату. Используется для расчёта потребления и начислений за период. | Показание 125,5 м³ на 01.03.2026 по счётчику воды. |
| **Дата показания (reading_date)** | Дата, на которую зафиксировано показание. Показания за разные даты образуют историю потребления по счётчику. | Показания на 01.01.2026 и 01.02.2026 — разница даёт расход за январь. |
| **Статус прибора** | Состояние счётчика в учёте: active (в работе), inactive (выведен из эксплуатации). Неактивные приборы не используются для новых начислений. | Счётчик снят — status = inactive. |

Смежные термины: финансовый субъект по счётчику — [financial.md](financial.md); владелец — [land.md](land.md); начисления — [accruals.md](accruals.md).
