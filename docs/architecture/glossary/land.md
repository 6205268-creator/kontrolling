# Глоссарий домена «Земля и владельцы (Land)»

Термины, связанные с земельными участками, владельцами и правами собственности в СТ.

| Термин | Определение | Пример использования в СТ |
|--------|-------------|----------------------------|
| **Владелец (Owner)** | Физическое или юридическое лицо, которое может владеть участками и приборами учёта. Тип задаётся полем `owner_type` (physical, legal). С СТ владелец не связан напрямую — только через участки. | Иванов И. И. (physical), исполком (legal) — оба могут быть владельцами участков. |
| **Земельный участок (LandPlot)** | Участок земли, принадлежащий одному СТ; характеризуется номером, площадью и опционально кадастровым номером. На участок создаётся один FinancialSubject типа LAND_PLOT для начислений и платежей. | Участок № 12, площадь 600 м², кадастровый номер 123-456; принадлежит СТ «Ромашка». |
| **Номер участка (plot_number)** | Уникальный в рамках СТ идентификатор участка (например, номер по генплану). | Участки № 1–150 в СТ «Ромашка». |
| **Площадь участка (area_sqm)** | Площадь участка в квадратных метрах; используется для расчёта взносов по тарифу за м². | Участок № 12 — 600 м²; при тарифе 0,05 BYN/м² начисление 30 BYN. |
| **Право собственности (PlotOwnership)** | Связь владельца с участком на определённый период с указанием доли. Один участок может иметь несколько владельцев (долевая собственность). | Иванов владеет участком № 12 с 01.01.2020, доля 1/1, is_primary = true. |
| **Доля собственности** | Доля владения участком: задаётся числителем и знаменателем (`share_numerator` / `share_denominator`). Сумма долей всех владельцев участка равна 1. | 1/1 — единоличное владение; 1/2 — половина участка (второй владелец тоже 1/2). |
| **Основной владелец (is_primary)** | Признак того, что владелец является членом СТ и несёт ответственность за членские взносы по данному участку. На одном участке только один владелец может иметь is_primary = true. | У участка № 12 основной владелец — Иванов; супруга вписана с долей 1/2, но is_primary = false. |
| **Период действия права (valid_from, valid_to)** | Даты начала и окончания записи о праве собственности. valid_to = null — право действует по сей день. Смена владельца оформляется закрытием старой записи (valid_to) и созданием новой. | Передача участка: у старой записи valid_to = 31.12.2024, у новой valid_from = 01.01.2025. |
| **Статус участка** | Состояние участка в учёте СТ: active (активный), vacant (свободный), archived (архивный). Влияет на отображение в списках и возможность начислений. | Участок № 99 — vacant, пока не оформлено право собственности. |
| **Кадастровый номер** | Государственный идентификатор земельного участка (опционально). Хранится в поле `cadastral_number`. | Кадастровый номер участка № 12 — 123-456 для сверки с госучётом. |
