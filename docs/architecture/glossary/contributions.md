# Глоссарий домена «Взносы»

| Термин | Определение | Пример использования в СТ |
|--------|-------------|----------------------------|
| **Взнос (целевой/членский)** | Обязательный платёж члена СТ: членский — на текущую деятельность, целевой — на конкретные цели (ремонт дороги, забор и т.д.). | Членский взнос 50 руб./мес.; целевой взнос на установку шлагбаума — 200 руб. с участка. |
| **Долг** | Просроченная задолженность по взносам: сумма не внесённых в срок платежей. | У участка №12 долг 150 руб. за III квартал 2024. |
| **Тариф** | Ставка взноса: фиксированная сумма с участка или расчёт от площади (руб./м²). | Тариф членского взноса 0,05 руб./м²; тариф с участка — 25 руб./мес. |
| **Распределение платежа** | Алгоритм зачисления поступивших денег: в какую очередь погашать долг, текущие взносы, пени. | Сначала пени, затем просроченный долг, затем текущий взнос за месяц. |
| **Пени** | Штраф за просрочку уплаты взносов по законодательству РБ (процент за каждый день просрочки). | Пени 0,1% в день от неуплаченной суммы по ст. 309 ГК РБ. |
