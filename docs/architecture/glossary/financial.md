# Глоссарий домена «Финансовое ядро»

Термины центрального финансового абстракции и балансов.

| Термин | Определение | Пример использования в СТ |
|--------|-------------|----------------------------|
| **FinancialSubject (финансовый субъект)** | Сущность, через которую ведутся все начисления и платежи; финансовая проекция бизнес-объекта (участок, счётчик, решение собрания). | У участка №12 один FinancialSubject типа LAND_PLOT; по нему считаются долг и оплаты. |
| **Баланс** | Разность между суммой начислений и суммой платежей по финансовому субъекту на дату. | Баланс участка №12 на 01.03.2026: +150 BYN (долг). |
| **subject_type** | Тип привязки финансового субъекта к бизнес-объекту: LAND_PLOT, WATER_METER, ELECTRICITY_METER, GENERAL_DECISION. | LAND_PLOT — взносы по участку; WATER_METER — расчёт по счётчику воды. |

*Документ будет дополняться по мере уточнения доменной терминологии.*
