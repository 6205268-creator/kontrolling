# Глоссарий домена «Начисления (Accruals)»

Термины, связанные с начислением взносов и видов платежей.

| Термин | Определение | Пример использования в СТ |
|--------|-------------|----------------------------|
| **Начисление (Accrual)** | Запись о начисленной сумме по финансовому субъекту за период; тип задаётся видом взноса (ContributionType). | Начисление членского взноса 50 BYN на участок №12 за март 2026. |
| **Вид взноса (ContributionType)** | Справочник типов начислений: членский, целевой и т.д. | Членский взнос — ежемесячно; целевой «Дорога» — разово. |
| **Период начисления** | period_start, period_end — интервал, за который начислена сумма. | Март 2026: period_start 2026-03-01, period_end 2026-03-31. |

Смежные термины (взносы, долг, пени, распределение платежа) — см. [contributions.md](contributions.md).

*Документ будет дополняться по мере уточнения доменной терминологии.*
