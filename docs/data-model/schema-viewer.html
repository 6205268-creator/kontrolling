Схема связей (таблицы можно перетаскивать)

Линия идёт от таблицы, где лежит поле (FK), к таблице, на которую ссылаются (PK). Подпись на линии — имя поля. Таблицы можно двигать мышью — расположение сохранится при обновлении страницы.

Сущности (EN → RU)

Cooperative
Садоводческое товарищество (СТ)
Owner
Владелец (физ./юр. лицо); к СТ — через участок
LandPlot
Земельный участок (cooperative_id); владение через PlotOwnership
PlotOwnership
Право/доля собственности: участок ↔ владелец, период, доля
Meter
Счётчик; привязан к владельцу
ContributionType
Вид взноса (справочник)
Payment
Платёж: участок, сумма, дата, статус
Accrual
Начисление: участок, вид взноса, сумма, период
MeterReading
Показание счётчика: значение, дата
