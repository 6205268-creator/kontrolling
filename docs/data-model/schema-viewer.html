Связи между полями (как в Access: это поле → то поле)

Каждая строка — одна связь. Поле в колонке «Поле (FK)» ссылается на поле «Поле (PK)» в другой таблице.

Схема связей (таблицы можно перетаскивать)

Линия идёт от таблицы, где лежит поле (FK), к таблице, на которую ссылаются (PK). Подпись на линии — имя поля. Таблицы можно двигать мышью — расположение сохранится при обновлении страницы.

Сущности (для справки)

Сущности (EN → RU):

Cooperative
Садоводческое товарищество (СТ)
Owner
Владелец (owner_type: физ. лицо, юр. лицо); с СТ не связан — к СТ через участок
LandPlot
Земельный участок (cooperative_id); владение только через PlotOwnership
PlotOwnership
Право/доля собственности: участок ↔ владелец, период (valid_from, valid_to), доля (1, 1/2, 1/3…)
Meter
Счётчик; привязан к владельцу (owner_id)
