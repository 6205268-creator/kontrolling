# Minimal entity set (Phase 1 — core)

**Минимальный набор сущностей (ядро, фаза 1)**  
Entity names in **English**; Russian meaning in comments for clarity.  
Имена сущностей на **английском**; русский смысл — в комментариях.

---

## 1. Cooperative

**RU:** Садоводческое товарищество (СТ). Одна запись = одно товарищество. Используется как tenant при мультитенантности.

| Attribute (EN)   | Type   | Comment (RU)                |
|-----------------|--------|-----------------------------|
| id              | string | Уникальный идентификатор СТ |
| name            | string | Название товарищества       |

---

## 2. Person

**RU:** Физическое лицо. Принадлежит **одному** СТ (не общая сущность для нескольких товариществ). Лицо и Член СТ — разные сущности; связь Person–Member 1:1.

| Attribute (EN)   | Type   | Comment (RU)                          |
|-----------------|--------|----------------------------------------|
| id              | string | Уникальный идентификатор физ. лица    |
| cooperative_id  | string | СТ, к которому относится лицо (FK)     |
| full_name       | string | ФИО                                    |
| contact_phone   | string | Телефон (опционально)                  |
| contact_email   | string | Электронная почта (опционально)        |

---

## 3. Member

**RU:** Член СТ. Связь с Person — **1:1** в рамках одного СТ: одно физическое лицо не может быть дважды членом одного товарищества. Один человек может владеть несколькими участками в том же СТ.

| Attribute (EN)   | Type   | Comment (RU)                     |
|-----------------|--------|-----------------------------------|
| id              | string | Уникальный идентификатор члена   |
| person_id       | string | Физ. лицо (FK), 1:1               |
| cooperative_id   | string | СТ (FK)                           |
| status          | string | Статус членства (активный и т.д.) |
| date_joined     | date   | Дата вступления                   |
| date_left       | date   | Дата выхода (опционально)         |

---

## 4. LandPlot

**RU:** Земельный участок. Принадлежит одному СТ.

| Attribute (EN)   | Type   | Comment (RU)                |
|-----------------|--------|-----------------------------|
| id              | string | Уникальный идентификатор    |
| cooperative_id  | string | СТ (FK)                     |
| plot_number     | string | Номер участка               |
| area_sqm        | number | Площадь, кв. м              |
| cadastral_number| string | Кадастровый номер (опц.)   |
| status          | string | Статус (активный и т.д.)   |

---

## 5. PlotOwnership

**RU:** Право собственности на участок. Связь «участок — собственник» с указанием типа права (индивидуальная, долевая, совместная) и доли. Один человек может иметь несколько участков; один участок — несколько собственников (долевая/совместная).

| Attribute (EN)     | Type   | Comment (RU)                              |
|--------------------|--------|-------------------------------------------|
| id                | string | Уникальный идентификатор права            |
| plot_id           | string | Участок (FK)                              |
| person_id         | string | Собственник — физ. лицо (FK)              |
| ownership_type    | string | Тип: индивидуальная, долевая, совместная |
| share_numerator   | number | Числитель доли (для долевой)              |
| share_denominator | number | Знаменатель доли (для долевой)            |
| date_acquired     | date   | Дата приобретения права                   |
| date_terminated   | date   | Дата прекращения (опционально)            |

---

## Relationships (связи)

| From (EN)   | To (EN)       | Cardinality | Comment (RU)                              |
|-------------|---------------|-------------|-------------------------------------------|
| Cooperative | Person        | 1 : N       | В СТ много физ. лиц                       |
| Cooperative | Member        | 1 : N       | В СТ много членов                        |
| Cooperative | LandPlot      | 1 : N       | В СТ много участков                      |
| Person      | Member        | 1 : 1       | Один человек — один член в данном СТ     |
| LandPlot    | PlotOwnership | 1 : N       | У участка может быть несколько прав       |
| Person      | PlotOwnership | 1 : N       | Один человек может владеть многими участками |
