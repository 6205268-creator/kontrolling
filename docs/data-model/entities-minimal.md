# Minimal entity set (Phase 1 — core)

**Минимальный набор сущностей (ядро, фаза 1)**  
Entity names in **English**; Russian meaning in comments for clarity.  
Имена сущностей на **английском**; русский смысл — в комментариях.

**Нормальная форма:** cooperative_id только у LandPlot. Связи Владелец ↔ СТ нет — к СТ идём через участок (LandPlot → Cooperative). Владение участком задаётся только через PlotOwnership (участок, владелец, период, доля); участок без владельца — нет записей PlotOwnership на нужную дату.

---

## 1. Cooperative

**RU:** Садоводческое товарищество (СТ). Одна запись = одно товарищество. Используется как tenant при мультитенантности.

| Attribute (EN)   | Type   | Comment (RU)                |
|-----------------|--------|-----------------------------|
| id              | string | Уникальный идентификатор СТ |
| name            | string | Название товарищества       |

---

## 2. Owner

**RU:** Владелец. Абстрактная сущность: владелец участка или счётчика может быть **физическим лицом**, **юридическим лицом** (напр. исполком) или участок может быть **без владельца** (нет записей PlotOwnership на нужную дату). С СТ владелец напрямую не связан — к товариществу идём через участок (LandPlot → Cooperative). Тип владельца задаётся в owner_type.

| Attribute (EN)   | Type   | Comment (RU)                          |
|-----------------|--------|----------------------------------------|
| id              | string | Уникальный идентификатор владельца     |
| owner_type      | string | Тип: physical (физ. лицо), legal (юр. лицо)… |
| name            | string | Наименование (ФИО или название орг.)  |
| contact_phone   | string | Телефон (опционально)                  |
| contact_email   | string | Электронная почта (опционально)        |

---

## 3. LandPlot

**RU:** Земельный участок. Принадлежит одному СТ (cooperative_id). Владение задаётся только через PlotOwnership (участок без владельца — нет записей на нужную дату).

| Attribute (EN)   | Type   | Comment (RU)                |
|-----------------|--------|-----------------------------|
| id              | string | Уникальный идентификатор    |
| cooperative_id  | string | СТ (FK)                     |
| plot_number     | string | Номер участка               |
| area_sqm        | number | Площадь, кв. м              |
| cadastral_number| string | Кадастровый номер (опц.)   |
| status          | string | Статус (активный и т.д.)   |

---

## 4. PlotOwnership

**RU:** Право/доля собственности на участок. Связь участок ↔ владелец с периодом (valid_from, valid_to) и долей (дробь: 1 = весь участок, 1/2 = половина, 1/3 = треть). По дате можно определить, кто владел участком и какой долей; допускается несколько владельцев в один период (доли в сумме = 1).

| Attribute (EN)     | Type   | Comment (RU)                                    |
|--------------------|--------|--------------------------------------------------|
| id                 | string | Уникальный идентификатор                        |
| land_plot_id       | string | Участок (FK)                                    |
| owner_id           | string | Владелец (FK)                                   |
| valid_from         | date   | Начало периода действия права                   |
| valid_to           | date   | Конец периода (опц.; пусто — действует по сей день) |
| share_numerator    | int    | Числитель доли (напр. 1)                        |
| share_denominator  | int    | Знаменатель доли (1 = вся доля, 2 = 1/2, 3 = 1/3) |

---

## 5. Meter

**RU:** Счётчик (прибор учёта). Привязан к **владельцу (Owner)** (owner_id). Принадлежность к СТ — через участки владельца или контекст учёта.

| Attribute (EN)     | Type   | Comment (RU)                       |
|--------------------|--------|-------------------------------------|
| id                 | string | Уникальный идентификатор счётчика  |
| owner_id           | string | Владелец счётчика (FK)             |
| number             | string | Номер счётчика                     |
| meter_type         | string | Тип: ЭЭ, вода, газ…               |
| installation_date  | date   | Дата установки (опц.)              |
| status             | string | Статус (в работе, выведен…)        |

---

## Relationships (связи)

| From (EN)      | To (EN)    | Cardinality | Comment (RU)                                                                 |
|----------------|------------|-------------|-------------------------------------------------------------------------------|
| Cooperative    | LandPlot   | 1 : N       | В СТ много участков                                                          |
| LandPlot       | PlotOwnership | 1 : N    | У участка много записей права/доли; по дате — кто владел и какой долей        |
| Owner          | PlotOwnership | 1 : N    | У владельца много записей по участкам и периодам                              |
| Owner          | Meter      | 1 : N       | У владельца может быть несколько счётчиков                                    |
