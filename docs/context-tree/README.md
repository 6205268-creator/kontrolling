# Дерево фич и пакеты контекста

Используй пакет контекста фичи, чтобы выполнять задачу с **минимально достаточным набором файлов** — без загрузки всего проекта. Это экономит токены и ускоряет ответы.

## Как пользоваться

1. **Задача привязана к фиче** (например: «Сделай фичу 13», «Доработай API участков»):
   - Открой `features/F13.md`.
   - Подгрузи в контекст **только** файлы из блока «Контекст» и секцию этой фичи из `docs/project-implementation.md`.
   - Выполняй шаги по плану.

2. **Задача без номера фичи**: по описанию определи блок/фичу → возьми пакет из `features/Fxx.md`.

3. **Общие принципы** всегда в силе: `docs/project-implementation.md` (заголовок + принципы), `.cursor/rules/project-conventions.mdc`.

## Структура

```
context-tree/
  README.md          — этот файл (дерево + как пользоваться)
  features/
    F01.md … F35.md  — пакет контекста для каждой фичи
```

В каждом `Fxx.md`: список файлов (контекст), правила, зависимости от других фич.
