**Версия:** 1
**Дата:** 2026-02-22

**Простыми словами:**
Реализована аутентификация пользователей через API: эндпоинт POST /api/v1/auth/login принимает username и password, проверяет credentials и возвращает JWT-токен. Добавлены зависимости для получения текущего пользователя из токена и проверки ролей. Все 7 тестов проходят.

**Роль в структуре проекта (технически):**
`backend/app/api/v1/auth.py` — router с эндпоинтом login. `backend/app/api/deps.py` — зависимости `get_current_user` (OAuth2PasswordBearer + JWT decode) и `require_role` (проверка роли). Интеграция с `AppUser` моделью и `security.py` (verify_password, create_access_token). Тесты в `test_api/test_auth.py` (7 тестов: успешный логин, wrong password, nonexistent user, inactive user, different roles). Основа для защищённых API endpoints (фичи 11+).
