## Выполнено

**Версия:** 1  
**Дата:** 2026-02-24  

**Простыми словами:**  
Включена история изменений для четырёх важных сущностей: права на участки (PlotOwnership), начисления (Accrual), платежи (Payment), расходы (Expense). При каждом создании, изменении или удалении такой записи в ту же транзакцию добавляется строка в отдельную таблицу истории с полным снимком данных и типом операции (insert/update/delete). Никаких новых библиотек — только встроенные события SQLAlchemy.

**Роль в структуре проекта (технически):**  
Модели истории в `backend/app/models/history.py` (PlotOwnershipHistory, AccrualHistory, PaymentHistory, ExpenseHistory); таблицы создаются миграцией `0009_add_history_tables.py`. Обработчики событий в `backend/app/db/history_events.py` регистрируются при старте приложения в `main.py`. Тесты в `tests/test_models/test_history.py`. Дальше можно добавить API для просмотра истории по entity_id или эндпоинты отчётов по аудиту.
