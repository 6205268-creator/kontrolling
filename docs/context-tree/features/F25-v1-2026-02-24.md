## Выполнено

**Версия:** 1
**Дата:** 2026-02-24

**Простыми словами:**
Создана страница «Владельцы» для просмотра списка собственников (физических и юридических лиц). Реализован поиск по имени и УНП, а также возможность добавления нового владельца через модальное окно. Страница интегрирована с backend API для загрузки и создания данных.

**Роль в структуре проекта (технически):**
- **Слой:** Frontend (Vue 3 + TypeScript + Pinia)
- **Зависимости:** F22 (Dashboard и навигация), F12 (API Owners CRUD)
- **Компоненты:**
  - `frontend/src/stores/owners.ts` — Pinia store для управления состоянием владельцев
  - `frontend/src/views/OwnersView.vue` — View-компонент с таблицей, поиском и модалкой создания
- **Интеграция:** API endpoints `/api/v1/owners/` и `/api/v1/owners/search`
- **Дальнейшее использование:** данные владельцев используются при создании участков (F24), начислений (F26), платежей (F27)
