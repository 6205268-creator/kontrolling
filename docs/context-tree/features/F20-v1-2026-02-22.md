# Фича 20: Frontend — Инициализация Vue 3 проекта

---
## Выполнено

**Версия:** 1
**Дата:** 2026-02-22

**Простыми словами:**
Создан frontend-проект на Vue 3 с TypeScript для системы учёта СТ. Настроена базовая структура: роутинг между страницами, хранилище состояния (Pinia) для аутентификации, сервис для API-запросов с автоматической подстановкой JWT-токена. Реализованы страница входа с формой логина и главная страница (dashboard) с навигацией по разделам системы. Сборка проекта проходит без ошибок.

**Роль в структуре проекта (технически):**
- **Слой:** Frontend (Vue 3 + TypeScript + Vite)
- **Зависимости:** F10 (Backend API auth) — эндпоинт `/api/v1/auth/login` для JWT-аутентификации
- **От неё зависит:** Все последующие frontend-фичи (F21–F26+) — страницы владельцев, участков, начислений, платежей, расходов, счётчиков, отчётов
- **Ключевые файлы:**
  - `src/services/api.ts` — axios instance с interceptor для JWT
  - `src/stores/auth.ts` — Pinia store (token, user, login, logout)
  - `src/router/index.ts` — vue-router с navigation guard
  - `src/views/LoginView.vue` — форма аутентификации
  - `src/views/DashboardView.vue` — главная страница с навигацией
  - `src/types/index.ts` — TypeScript интерфейсы для всех сущностей
---

**Цель:** Vue 3 + TypeScript + Vite + Pinia, router, api.ts, auth store.
