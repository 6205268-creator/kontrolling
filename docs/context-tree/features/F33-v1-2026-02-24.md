# Фича 33: Деплой — Docker контейнеризация

---
## Выполнено

**Версия:** 2
**Дата:** 2026-02-27

**Простыми словами:**
Настроили запуск всего проекта в Docker: база данных PostgreSQL, backend на FastAPI и frontend на Vue 3 собираются в контейнеры и запускаются одной командой. Миграции применяются автоматически, frontend отдаётся через Nginx.

**Роль в структуре проекта (технически):**
- **Слой инфраструктуры:** Docker-контейнеризация для развёртывания приложения
- **Зависимости:** F01 (backend структура), F20 (frontend структура)
- **От этой фичи зависит:** F34 (CI/CD), F35 (E2E тесты)
- **Файлы:**
  - `backend/Dockerfile` — Python 3.11-slim, uvicorn, авто-миграции Alembic
  - `frontend/Dockerfile` — multi-stage (Node 20 → Nginx alpine)
  - `frontend/nginx.conf` — проксирование /api на backend, SPA роутинг
  - `docker-compose.yml` — 3 сервиса + volume для PostgreSQL
  - `backend/.dockerignore`, `frontend/.dockerignore` — исключения сборки
  - `DEPLOY.md` — документация для разработчиков
