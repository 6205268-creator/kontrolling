# Фича 12: API endpoints — Owners (CRUD)

---
## Выполнено

**Версия:** 1
**Дата:** 2026-02-22

**Простыми словами:**
Реализован полный CRUD для владельцев (физических и юридических лиц) с поиском по имени и УНП. Теперь можно создавать, просматривать, обновлять и удалять владельцев. Поиск работает по частичному совпадению. Права доступа: создание и обновление — казначей и админ, удаление — только админ.

**Роль в структуре проекта (технически):**
- **Слой:** API (endpoints) + Service Layer
- **Зависимости:** F02 (Owner модель), F09 (Owner схемы), F10 (аутентификация и роли)
- **Интеграция:** 
  - Использует модели `app.models.owner.Owner`
  - Использует схемы `app.schemas.owner.OwnerCreate/OwnerUpdate/OwnerInDB`
  - Использует систему аутентификации через `app.api.deps.get_current_user` и `require_role`
  - Подключен в `app.main.py` как `/api/v1/owners`
- **Что использует:** JWT-токены, разграничение по ролям (admin/treasurer)
- **Что от неё зависит:** F13 (LandPlots — для создания владений участками), F14/F15 (Accruals/Payments — для платежей от владельцев)

---

**Цель:** CRUD по Owner, поиск владельцев.

## Реализация

### Созданные файлы:

1. **`backend/app/services/owner_service.py`** — сервисный слой с функциями:
   - `create_owner()` — создание владельца
   - `get_owner()` — получение по ID
   - `get_owners()` — получение списка (пагинация)
   - `update_owner()` — обновление по ID
   - `delete_owner()` — удаление по ID
   - `search_owners()` — поиск по имени/tax_id (ilike)

2. **`backend/app/api/v1/owners.py`** — API endpoints:
   - `GET /api/v1/owners/` — список (пагинация skip/limit)
   - `GET /api/v1/owners/{id}` — получение по ID
   - `POST /api/v1/owners/` — создание (treasurer, admin)
   - `PATCH /api/v1/owners/{id}` — обновление (treasurer, admin)
   - `DELETE /api/v1/owners/{id}` — удаление (admin)
   - `GET /api/v1/owners/search?q={query}` — поиск

3. **`backend/tests/test_api/test_owners.py`** — 13 тестов:
   - Создание physical/legal
   - Валидация owner_type
   - Получение списка и по ID
   - Поиск по имени и tax_id
   - Обновление
   - Удаление с проверкой прав
   - Обработка 404 и 403
