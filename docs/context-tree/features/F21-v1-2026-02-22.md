## Выполнено

**Версия:** 1
**Дата:** 2026-02-22

**Простыми словами:**
Создана страница входа в систему с формой логина и пароль. Пользователь вводит credentials, система проверяет их через backend API, получает JWT-токен и сохраняет его в localStorage. При успешном входе — редирект на dashboard. При ошибке — показывается сообщение. Токен автоматически добавляется ко всем запросам через axios-интерцептор.

**Роль в структуре проекта (технически):**
- Frontend слой аутентификации: Vue 3 + Pinia store для управления состоянием пользователя
- Интеграция с backend через axios с JWT interceptor (автоматическое добавление токена в заголовки)
- Роутинг с navigation guard для защиты приватных роутов
- Типизация через TypeScript (интерфейсы User, Token в types/index.ts)
- Зависимости: F20 (инициализация проекта), F10 (backend auth endpoint)
