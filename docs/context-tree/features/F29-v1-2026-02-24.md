## Выполнено

**Версия:** 1
**Дата:** 2026-02-24

**Простыми словами:**
Создана страница «Отчёт о движении средств» для просмотра финансовой статистики за выбранный период. Отчёт показывает общую сумму начислений, платежей и расходов, а также чистый баланс (разницу между ними). Реализован экспорт данных в CSV и автоматический выбор текущего месяца при открытии страницы.

**Роль в структуре проекта (технически):**
- **Слой:** Frontend (Vue 3 + TypeScript + Pinia)
- **Зависимости:** F22 (Dashboard и навигация), F28 (Отчёт по должникам — общий store reports), F19 (API reports/cash-flow endpoint)
- **Компоненты:**
  - `frontend/src/views/CashFlowReportView.vue` — View-компонент с формой выбора периода и отображением отчёта
  - `frontend/src/stores/reports.ts` — существующий Pinia store с методом `fetchCashFlowReport()`
  - `frontend/src/router/index.ts` — добавлен роут `/reports/cash-flow`
- **Интеграция:** API endpoint `/api/v1/reports/cash-flow` с параметрами cooperative_id, period_start, period_end
- **Дальнейшее использование:** отчёт используется для финансового анализа и принятия решений председателями и казначеями СТ
