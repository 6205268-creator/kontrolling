## Выполнено

**Версия:** 1  
**Дата:** 2026-02-24  

**Простыми словами:**  
Реализована страница «Создание участка»: пользователь заполняет номер участка, площадь, при необходимости кадастровый номер и статус, выбирает СТ (админ — из списка, остальные видят своё СТ). Владельцев можно добавлять по поиску (имя или УНП), для каждого указывается доля (числитель/знаменатель), дата начала владения и отметка «Основной владелец (член СТ)». Форма проверяет, что есть хотя бы один владелец, ровно один основной и что сумма долей равна 1. После отправки данные уходят на бэкенд, при успехе выполняется переход к списку участков.

**Роль в структуре проекта (технически):**  
View-слой фронтенда (Vue 3): `frontend/src/views/LandPlotCreateView.vue`. Использует `api` (axios), `useAuthStore` для роли и cooperative_id, роут `land-plots/create` уже был в `router/index.ts`. Обращается к API: `GET cooperatives/`, `GET owners/search?q=`, `POST land-plots` с телом участка и массивом ownerships. Зависит от бэкенда F23 (создание участка с владельцами), F13/F12 (владельцы, кооперативы). Дальше от этой фичи тянется использование списка участков на `/land-plots` и последующие фичи редактирования/просмотра участка.
