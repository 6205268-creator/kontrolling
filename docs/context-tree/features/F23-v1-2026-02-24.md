## Выполнено

**Версия:** 1  
**Дата:** 2026-02-24  

**Простыми словами:**  
На странице «Участки» теперь показывается таблица всех земельных участков: номер, площадь, статус и количество владельцев. Администратор может фильтровать список по садовому товариществу; председатель и бухгалтер видят только участки своего СТ. Кнопка «Добавить участок» ведёт на форму создания (фича 24). Данные подгружаются с бэкенда при открытии страницы.

**Роль в структуре проекта (технически):**  
Фича реализована во фронтенде (Vue 3): типы `LandPlot`/`LandPlotWithOwners` в `types/index.ts`, Pinia-store `stores/landPlots.ts` с `fetchPlots(cooperativeId?)` и `createPlot(data)`, представление `views/LandPlotsView.vue` с таблицей, фильтром по СТ (для admin через API `cooperatives/`) и вызовом `fetchPlots` при монтировании. Опирается на API `GET /api/v1/land-plots/` (F13/F22) и роут `/land-plots` из роутера. Дальше от неё тянется фича 24 (форма создания участка использует тот же store и типы).
