Настройки расчета пени

Данный диалог предназначен для редактирования настроек расчета пени (рис. 1).

Рис. 1. Настройки расчета пени

Рассмотрим поля, доступные для заполнения:

  • Рассчитывать пени - признак расчета пени.

  • Настройка расчета для членских и целевых взносов

    • Процент пени - указывается процент для расчета пени.

    • Дата начала расчета - указывается дата начала расчета пени.

После нажатия на кнопку ОК изменения будут сохранены.
