Настройки блока

Данный диалог содержит в себе настройки блока СНТ.

Рис. 1. Диалог "Настройки блока"

Рассмотрим более подробно каждый из пунктов:

  • Источник данных - указывается источник данных для построения отчетов и расчета сальдо. Доступны два варианта:

    • ЖХО - источником данных является журнал хозяйственных операций. Используется по умолчанию.

    • База данных - данный вариант в текущий момент находится на этапе тестирования и пользователям не доступен.

  • Группировка участков при выборе - настройки позволяет изменить вид списка участков при выборе в бланках.

    • нет - группировки нет. Все участки выведен в единый список. Сортировка осуществляется по значению из настройки Сортировка участков по.

    • по улицам - осуществляется группировка по улицам. При выборе необходимо сначала выбрать улицу и только потом участок.

  • Сортировка участков по - указывается вариант сортировки участков при выборе в бланках.

    • числу в № - по числовой части номера участка.

    • полному № - по полному номеру участка

    • Ф.И.О. владельца - по Ф.И.О. владельцев участков.

  • Выводить улицу в отчетах - настройка позволяет убрать название улицы в столбце Адрес в отчетах.

  • Способ ввода показаний - указывается способ ввода показаний ИПУ.

    • Один раз в месяц - показания вносятся раз в месяц.

    • Несколько раз в месяц - показания вносятся несколько раз в месяц. Изменяется внешний вид таблицы показаний в справочнике ИПУ и таблицы ввода показаний в бланке 3.01.

  • Директория выгрузки - указывается директория выгрузки реестров начислений и отчетов.

  • Помнить предыдущий выбор в графах множественного выбора при выключенном режиме выделения - если опция выключена, то при отключенном режиме выделения в графе множественного выбора будет выбрана только запись под курсором. При включенной опции предыдущий выбор сбрасываться не будет и будет происходить накопление выбранных объектов.

  • Для граф выбора улиц и участков показывать только связанные улицы/участки - если опция включена, то при выборе в графах множественного выбора улиц или участков будут отображены только те улицы или участки, которые связаны друг с другом. При выключенной опции будут отображаться все улицы или участки..

  • Синхронизация с сайтом - опция включает алгоритмы работы с сайтом. Временно недоступна.

  • Профиль работы с сайтом - профиль работы с сайтом. Временно недоступен.

  • Быстрая выгрузка и загрузка - опция включает алгоритмы быстрой работы с сайтом. Временно недоступна.

  • Диап. объемов потребления ЭЭ - опция включает расчет с учетом диапазонов объемов потребления ЭЭ.
