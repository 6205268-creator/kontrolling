Возможности блока СНТ.

  • Выполнение расчётов по начислению и выгрузке:

    • начисление членских и целевых взносов. Возможность начисления сумм взносов на основе сумм из сметы.

    • начисление по показаниям индивидуальных приборы учёта потребления электроэнергии (1, 2 и 3 тарифные зоны),  воды и газа. Возможно начисление платежей без ввода показаний, учет социальной нормы электроэнергии.

    • начисление по показаниям общих приборов учёта потребления электроэнергии.

    • начисление пени.

    • выгрузка реестров начислений для передачи в банк.

  • Платёжные документы:

    • формирование и вывод на печать различных форм платежных документов.

    • формирование двумерных штрих-кодов(QR-кодов).

    • отправка платежных документов на электронную почту в различных форматах.

    • настраиваемые шаблоны полей платежных документов

  • Касса:

    • различные виды внесения оплат и их учет. Импорт оплат из Клиент-Банка.

    • кассовые документы.

  • Ведение смет (через дополнительный блок Сметы)

  • Расчет земельного налога:

    • для неприватизированных участков

    • за земли общего пользования (ЗОП)

  • Внесение исходных остатков по участкам для взносов и поставляемых услуг.

  • Учет информации по участкам и собственникам.

  • Различные формы отчетов.
