Бланк "Реестр внесенных оплат"

Данный бланк предназначен для отображения внесенных оплат с возможностью выбора счета, статьи и детализации (рис. 1).

Рис. 1. Бланк "Реестр внесенных оплат"

Бланк содержит графы множественного выбора. Подробное описание работы с данными графами находится тут.

Рассмотрим поля, доступные для заполнения:

  • Счета для отчета - указывается, по каким счетам формировать журнал - Касса и расчетный счет, Только касса, Только расчетный счет.

  • Тип отчёта - указывается тип отчёта. Доступны свернутый и развернутый отчёты. В свернутом отчете все оплаты по участку за период будут выведены в столбцы общими суммами по взносам(рис. 2). В развернутом варианте все оплаты за период будут выведены построчно.

  • Улицы — выбираются улицы, по которым необходимо выполнить расчет. Возможен выбор произвольного количества улиц. Если оставить графу пустой, то отбор будет осуществляться по всем улицам.

  • Участки — выбираются участки, по которым необходимо выполнить расчет. Возможен выбор произвольного количества участков. Если оставить графу пустой, то отбор будет осуществляться по всем участкам.

  • Взносы — выбираются взносы, по которым необходимо выполнить расчет. Возможен выбор произвольного количества взносов. Если оставить графу пустой, то отбор будет осуществляться по всем взносам.

  • Период с ... по ... - указывается период формирования ведомости.

  • выгрузить ... в формате ... - позволяет выгрузить на диск таблицу через указанное приложение в указанном формате.

  • кнопка Настройка вывода столбцов - позволяет настроить ширину и вывод столбцов. Данная настройка доступна не для всех типов отчёта.

  • вставлять пустую строку после оплат по участку - данная опция позволяет вставлять пустую строку после оплат по каждому участку для визуального разделения.

  • выводить адрес и Ф.И.О. только для первой оплаты по участку - данная опция позволяет выводить адрес и Ф.И.О. только у первой оплаты по участку. Далее эти поля не заполняются.

  • выводить Ф.И.О. владельца, при котором была внесена оплата - данная опция позволяет группировать оплаты с учетом периода членства владельцев участков. Если у участка несколько действующих владельцев, то будут выведены Ф.И.О. первого найденного.

После заполнения полей, влияющих на расчет, необходимо пересчитать бланк. Для этого нажмите кнопку на панели инструментов, клавишу F9, или комбинацию клавиш Ctrl + Enter.

После пересчета на экране отобразится таблица с внесенным оплатами (рис. 2).
