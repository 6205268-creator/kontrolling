Справочник тарифов

Данный справочник содержит в себе тарифы взносов(рис. 1).

Для различных типов взносов и методов расчета предусмотрены отличающиеся друг от друга редакторы тарифов. Рассмотрим возможные варианты:

  • Справочник тарифов на членские и целевые взносы, воду, газ и отопление.

    Рассмотрим справочник на примере справочника тарифов на членские взносов (рис. 1).

    Рис. 1. Справочник тарифов на членские и целевые взносы, воду, газ и отопление.

    Рассмотрим поля, доступные для заполнения:

    • Тариф - указывается тариф или сумма фиксированного взноса(для взносов с методом расчета Фикс. взнос).

    • Дата начала - дата начала действия тарифа.

    • Дата конца - дата окончания действия тарифа.

  • Справочник тарифов на электроэнергию

    Справочник тарифов на электроэнергию содержит в себе три подсправочника (рис. 3).

    Рис. 3. Выбор справочника тарифов на электроэнергию.

    После выбора откроется соответствующий справочник тарифов на электроэнергию. Рассмотрим справочник тарифов на электроэнергию на примере трехтарифного справочника (рис. 4).

    Рис. 4. Трехтарифный справочник тарифов на электроэнергию.

    Рассмотрим поля, доступные для заполнения:

    • Тариф Т... - тарифы для соответствующих тарифных зон. Количество доступных тарифов зависит от тарифности взноса.

    • Дата начала - дата начала действия тарифа.

    • Дата конца - дата окончания действия тарифа.

    • Признак - признак тарифа - сельский или обычный. Данный признак учитывается при расчете начислений за электроэнергию. Данный признак должен совпадать у тарифа и ИПУ, по которому ведётся расчет.

Дата окончания действия предыдущего тарифа не должна пересекаться с датой начала действия нового тарифа. Например, дата окончания предыдущего тарифа 30.06.2015. Значит дата начала действия нового тарифа должна быть 01.07.2015. В противном случае возможен некорректный выбор тарифа.
