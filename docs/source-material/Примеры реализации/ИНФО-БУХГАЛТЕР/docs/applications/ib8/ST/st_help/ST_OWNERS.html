Справочник владельцев

Данный справочник содержит в себе данные по владельцам (рис. 1).

Рис. 1. Редактирование записи

Рассмотрим поля, доступные для заполнения:

  • Ф.И.О. - фамилия, имя и отчество владельца.

  • Адрес регистрации - адрес регистрации владельца.

  • Адрес фактический - фактический адрес владельца.

  • Адрес почтовый - почтовый адрес владельца.

  • Дата рождения - дата рождения владельца.

  • Место рождения - место рождения владельца.

  • Паспортные данные - паспортные данные владельца.

  • Телефон - номер телефона владельца.

  • Моб. тел. 1 - номер мобильного телефона.

  • Моб. тел. 2 - номер дополнительного мобильного телефона.

  • E-mail - адрес электронной почты, на который будут отправляться квитанции.

  • Дата принятия - дата принятия в товарищество.

  • № протокола - номер протокола о принятии в товарищество.

  • Дата выбытия - дата выбытия из товарищества.
