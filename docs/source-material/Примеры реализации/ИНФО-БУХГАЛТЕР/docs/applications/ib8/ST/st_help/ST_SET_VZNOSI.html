Установка целевых взносов

Данная опция позволяет устанавливать целевые взносы для участков по произвольным диапазонам(рис. 1).

Рис. 1. Установка целевых взносов

Диалог содержит графы множественного выбора. Подробное описание работы с данными графами находится тут.

Рассмотрим доступные для ввода поля:

  • Улицы - выбираются улицы, по которым необходимо выполнить установку. Возможен выбор произвольного количества улиц. Если оставить графу пустой, то отбор будет осуществляться по всем улицам.

  • Участки - выбираются участки, по которым необходимо выполнить установку. Возможен выбор произвольного количества участков. Если оставить графу пустой, то отбор будет осуществляться по всем участкам.

  • Взносы - выбираются взносы, которые необходимо установить. Возможен выбор произвольного количества взносов. Если оставить графу пустой, то отбор будет осуществляться по всем взносам.

  • Признак - выбирается признак участков. Если ничего не выбрано, то установка будет производиться по всем типам участков. В противном же случае установка будет производиться только по участкам, где владельцы являются членами или не членами СНТ.

  • Действие - выбирается действие. Взносы можно добавить или удалить.

После нажатия на кнопку OK будет произведена установка целевого взноса по выбранному диапазону.
