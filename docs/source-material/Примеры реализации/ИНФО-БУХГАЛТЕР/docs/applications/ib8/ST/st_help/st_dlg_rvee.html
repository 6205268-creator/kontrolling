Диапазоны объемов потребления ЭЭ

Данный справочник содержит данные о диапазонах потребления электроэнергии. Данные диапазоны учитываются при расчете сумм начислений за электроэнергию, если включена соответствующая опция в настройках блока (рис. 1).

Рис. 1. Справочник "Диапазоны объемов потребления ЭЭ"

В справочнике присутствуют следующие поля:

  • Название — наименование диапазона.

  • Код — уникальный код(идентификатор) диапазона.

  • С объема — объем, с которого начинается диапазон(включительно).

  • По объем - объем, на котором заканчивается диапазон(включительно). Если стоит значение 0(ноль), то конечный объем не ограничен.
